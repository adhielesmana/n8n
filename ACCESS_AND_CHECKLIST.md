## 📋 Setup Checklist

### **Phase 1: Get API Keys** ⏱️ 10 minutes

- [ ] **Meta API**
  - [ ] Go to: https://developers.facebook.com/
  - [ ] Create app or select existing
  - [ ] Copy: App ID
  - [ ] Copy: App Secret
  - [ ] Copy: Page Access Token
  - [ ] Copy: Page ID
  - [ ] Copy: Instagram Business Account ID

- [ ] **Claude API**
  - [ ] Go to: https://console.anthropic.com/
  - [ ] Create API key
  - [ ] Copy: `sk-ant-...`

- [ ] **Twilio (WhatsApp)**
  - [ ] Go to: https://www.twilio.com/
  - [ ] Create account
  - [ ] Copy: Account SID
  - [ ] Copy: Auth Token
  - [ ] Copy: WhatsApp Number

### **Phase 2: Add Credentials to n8n** ⏱️ 5 minutes

In n8n → Settings:

- [ ] Add Meta API credential
  - [ ] Name: "Meta API"
  - [ ] Paste App ID
  - [ ] Paste App Secret
  - [ ] Paste Page Access Token
  - [ ] Save

- [ ] Add Claude credential
  - [ ] Name: "Claude API"
  - [ ] Paste API Key
  - [ ] Save

- [ ] Add Twilio credential
  - [ ] Name: "Twilio"
  - [ ] Paste Account SID
  - [ ] Paste Auth Token
  - [ ] Save

### **Phase 3: Create Workflows** ⏱️ 20 minutes

#### **Workflow 1: Instagram**
- [ ] Create new workflow: "Message Bot - Instagram"
- [ ] Add Webhook node (path: `instagram-messages`)
- [ ] Copy webhook URL
- [ ] Add Set node (extract sender_id, message_text)
- [ ] Add Claude node (generate reply)
- [ ] Add Instagram node (send reply)
- [ ] Test with sample message
- [ ] Activate

#### **Workflow 2: WhatsApp**
- [ ] Create new workflow: "Message Bot - WhatsApp"
- [ ] Add Webhook node (path: `whatsapp-messages`)
- [ ] Copy webhook URL
- [ ] Add Set node (extract phone, message)
- [ ] Add Claude node (generate reply)
- [ ] Add Twilio node (send WhatsApp)
- [ ] Test with sample message
- [ ] Activate

### **Phase 4: Register Webhooks** ⏱️ 10 minutes

#### **In Meta App Dashboard**
- [ ] Go to: https://developers.facebook.com/apps/YOUR_APP_ID/
- [ ] Messenger → Webhooks
- [ ] Paste Instagram webhook URL
- [ ] Set verify token: `n8n_bot_secure_2024`
- [ ] Subscribe to: `messages`, `messaging_postbacks`
- [ ] Save

#### **In Twilio Console**
- [ ] Go to: https://console.twilio.com/
- [ ] WhatsApp Sandbox
- [ ] Paste WhatsApp webhook URL
- [ ] Save

### **Phase 5: Test & Deploy** ⏱️ 10 minutes

- [ ] Send test message to Instagram
- [ ] Verify bot replies in 1-2 seconds
- [ ] Send test message to WhatsApp
- [ ] Verify bot replies in 1-2 seconds
- [ ] Check n8n Executions tab
- [ ] Both workflows show ✅ Activated
- [ ] Monitor logs for errors

## 🧪 Testing Commands

### **Test Instagram Webhook**
```bash
curl -X POST http://localhost:5678/webhook/instagram-messages \
  -H "Content-Type: application/json" \
  -d '{
    "messaging": [{
      "sender": {"id": "test_user_123"},
      "message": {"text": "Hello bot!"}
    }]
  }'
```

### **Test WhatsApp Webhook**
```bash
curl -X POST http://localhost:5678/webhook/whatsapp-messages \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [{
      "from": "+1234567890",
      "text": {"body": "Hello bot!"}
    }]
  }'
```

### **Check Docker Status**
```bash
docker-compose -f docker-compose.local.yml ps
```

### **View Logs**
```bash
docker-compose -f docker-compose.local.yml logs -f n8n
```

## ❌ Troubleshooting Quick Guide

| Issue | Check |
|-------|-------|
| Bot not replying | ✓ Webhook URL correct? ✓ Credentials valid? ✓ Activation ON? |
| Webhook error | ✓ n8n running? ✓ Docker status? ✓ Path correct? |
| Claude not working | ✓ API key valid? ✓ Credential saved? ✓ Token limit? |
| Meta not receiving | ✓ Verify token set? ✓ Permissions enabled? ✓ App approved? |
| Twilio timeout | ✓ Account SID correct? ✓ Auth token valid? ✓ WhatsApp enabled? |
